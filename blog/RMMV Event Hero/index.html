RMMV Event Hero

Нян!

По просьбе Петра (с «нейтралки») быстро родился такой вот небольшой плагин — «MUR Event Hero».

Если внутри события (event) задать комментарий вида <hero: ID>, где ID это номер (ID) героя в системе, то в этот event скопируется графика из этого героя.

Например, комментарий:

Означает, что как только будет активирован данный шаг события, будет загружена графика из героя с номером 1:

Шагов может быть неограниченное количество, и в каждом может быть указан свой <hero>.

Если указан неправильный ID героя, то консоль получим сообщение об ошибке, но игра не будет «блокироваться», просто не загрузится новая графика.

Если у героя не задана картинка, то и у события тоже вместо картинки будет пустота.

Посмотреть этот плагин в действии, а также некоторые другие можно на моём online-полигоне :3

Скачать свежую версию как всегда можно на github.
