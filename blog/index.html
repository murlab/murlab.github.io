30 Января 2018
RMMV Event Hero

Нян!

По просьбе Петра (с «нейтралки») быстрородился такой вот небольшой плагин — «MUR Event Hero».

Если внутри события (event) задать комментарий вида <hero: ID>, где ID это номер (ID) героя в системе, то в этот event скопируется графика из этого героя.
